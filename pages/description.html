Project Description

This project explores the development of an interactive 2D dungeon-crawler educational game designed for middle school students. The goal is to teach science and math concepts through engaging puzzle-based gameplay.

Players explore themed maps, solve puzzles tied to real curriculum topics, and earn progress by demonstrating understanding. The design blends classic game mechanics with educational goals, aiming to improve engagement and retention through play-driven learning.
